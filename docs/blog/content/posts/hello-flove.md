This is the flove blog — built with [Hugo](https://gohugo.io/), served static
from `flove.org/blog/`, with zero JavaScript and one small stylesheet.

Posts live in `blog-src/content/posts/`. Write Markdown, run `./build-blog.sh`,
commit the generated `/blog`, and it goes live on the next update-web pass.

_slow it · flow it · love it._
